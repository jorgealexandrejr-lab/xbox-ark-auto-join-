<a id="portuguese"></a>
## 🇧🇷 Português (Brasileiro)

Uma extensão do Google Chrome (Manifest V3) projetada para automatizar o processo de entrada em servidores lotados no **Xbox Remote Play** e **Xbox Cloud Gaming**, especificamente otimizada para o jogo **ARK: Survival Ascended**.

O script simula as entradas de botões do controle do Xbox (pressionando **X** para tentar se conectar e **B** para fechar a mensagem de "servidor cheio") em um loop contínuo e inteligente com tempos configuráveis.

### Como Instalar (Modo de Desenvolvedor)

Como esta extensão está hospedada de forma independente no GitHub, você precisa instalá-la usando o Modo de Desenvolvedor do Chrome. É super simples e leva menos de 1 minuto:

#### Passo 1: Baixar a Extensão
1. Clique no botão verde **Code** (no topo desta página) e selecione **Download ZIP**.
2. Extraia o arquivo `.zip` baixado em qualquer pasta permanente no seu computador (por exemplo, em documentos ou na área de trabalho).

#### Passo 2: Carregar no Google Chrome
1. Abra o Google Chrome.
2. Acesse a página de extensões digitando `chrome://extensions/` na barra de endereços (ou clique no menu de três pontos $\rightarrow$ **Extensões** $\rightarrow$ **Gerenciar extensões**).
3. No canto superior direito, ative a chave **"Modo do desenvolvedor"** (Developer Mode).
4. No canto superior esquerdo que irá surgir, clique no botão **"Carregar sem compactação"** (Load unpacked).
5. Selecione a pasta que você extraiu no Passo 1 (certifique-se de selecionar a pasta que contém o arquivo `manifest.json`).
6. **Pronto!** A extensão está ativa. O ícone do implante azul do ARK deve aparecer na sua lista de extensões.

### 🕹️ Como Usar

1. Acesse o site oficial do Xbox Cloud Gaming / Remote Play: [xbox.com/play](https://www.xbox.com/play) ou [xbox.com/remoteplay](https://www.xbox.com/remoteplay).
2. Inicie a transmissão do seu console e abra a lista de servidores no **ARK**.
3. O painel flutuante **Xbox Auto-Join** aparecerá automaticamente no canto superior direito da tela.
4. Ajuste os tempos nos campos se necessário:
   - **Espera após X (segundos):** Tempo que o jogo leva para tentar conectar e exibir a mensagem de erro (recomendado: 4.0s).
   - **Espera após B (segundos):** Tempo que o jogo leva para fechar a janela de erro e voltar à lista (recomendado: 2.0s).
5. Clique em **Iniciar** e a automação começará a rodar!
6. **Parada de Emergência:** A qualquer momento, clique em **Parar** na interface ou pressione a tecla `ESC` ou `Espaço` no teclado para parar o loop instantaneamente.

<a id="english"></a>
## 🇺🇸 English

A Google Chrome extension (Manifest V3) designed to automate the process of joining full servers on **Xbox Remote Play** and **Xbox Cloud Gaming**, specifically optimized for the game **ARK: Survival Ascended**.

The script simulates Xbox controller button presses (pressing **X** to attempt connection and **B** to dismiss the "server is full" popup) in a continuous, smart loop with customizable delays.

### How to Install (Developer Mode)

Since this extension is independently hosted on GitHub, you need to load it using Chrome's Developer Mode. It is very simple and takes less than a minute:

#### Step 1: Download the Extension
1. Click the green **Code** button at the top of this page and select **Download ZIP**.
2. Extract the downloaded `.zip` file into any permanent folder on your computer (e.g., Documents or Desktop).

#### Step 2: Load into Google Chrome
1. Open Google Chrome.
2. Go to the extensions page by typing `chrome://extensions/` in the address bar.
3. In the top-right corner, enable the **"Developer Mode"** toggle.
4. In the top-left corner, click the **"Load unpacked"** button.
5. Select the folder you extracted in Step 1 (make sure to select the directory containing the `manifest.json` file).
6. **Done!** The extension is now active. The blue ARK specimen implant icon should appear in your extensions list.

### 🕹️ How to Use

1. Go to the official Xbox Cloud Gaming / Remote Play site: [xbox.com/play](https://www.xbox.com/play) or [xbox.com/remoteplay](https://www.xbox.com/remoteplay).
2. Start your console stream and navigate to the server list in **ARK**.
3. The floating **Xbox Auto-Join** panel will automatically appear in the top-right corner of your screen.
4. Adjust the delays if necessary:
   - **Delay after X (seconds):** The time the game takes to attempt a connection and show the error message (recommended: 4.0s).
   - **Delay after B (seconds):** The time the game takes to close the error window and return to the list (recommended: 2.0s).
5. Click **Iniciar** (Start) to begin!
6. **Emergency Stop:** At any point, click **Parar** (Stop) on the UI or press the `ESC` or `Space` key on your keyboard to instantly stop the loop.

     - **Espera después de B (segundos):** El tiempo que tarda el juego en cerrar la ventana de error y regresar a la lista (recomendado: 2.0s).

6. **Parada de emergencia:** En cualquier momento, haz clic en **Parar** en la interfaz o presiona la tecla `ESC` o `Espacio` en tu teclado para detener el bucle instantáneamente.
